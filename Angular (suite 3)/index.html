Exercice 1

Affichez dans une liste les livres, de manière à obtenir... Hugo, Victor : Les Misérables. Zola, Émile : L'Assommoir. Barjavel, René : La Nuit des temps. Verne, Jules : Le Tour du monde en quatre-vingts jours. Triez maintenant ces œuvres par ordre croissant des noms d'auteur, de manière à obtenir... Barjavel, René : La Nuit des temps. Hugo, Victor : Les Misérables. Verne, Jules : Le Tour du monde en quatre-vingts jours. Zola, Émile : L'Assommoir. Créez maintenant un filtre permettant d'afficher uniquement les œuvres ayant été publiées après la date saisie dans le champ de formulaire. Par exemple, si la date saisie est 1900, le seul ouvrage à apparaître est la Nuit des temps. Les deux filtres doivent être appliqués simultanément.
