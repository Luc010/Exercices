Exercices

Exercice 1

Créer une formulaire qui affiche deux zones de textes.

Dans la première zone de texte saisir : une chaine de caractère de plus de 5 caractères

Afficher dans la deuxième zone le 4ème caractère de cette chaine.

Exercice 2

Dans une zone de texte saisir une date sous le format jour/mois/année (15/08/2019)

Afficher dans trois zones de texte distinctes le jour , le mois et l'année

Exercice 3

On vous donne un nom de domaine dans une zone de texte : www.afpa.com

Afficher son extension.

Exercice 4

Soit deux zones de texte, saisir un texte en miniscule dans la premiere zone de texte (text1), cliquer sur le bouton convertir pour afficher dans la deuxième zone de texte le texte en MAJUSCULE.

Exercice 5

Vous avez une page avec un formulaire et les champs suivants zone de texte, textarea, radio checkbox,liste de valeurs Écrivez un script qui va valider le formulaire. Si le champ est vide, un message d'erreur est affiché à côté du champ et le formulaire n'est pas envoyé. Si le champ contient des données, le formulaire peut être envoyé.

Exercice 6

Définir un objet 'member' (membre) avec les attributs 'id' (identifiant), 'name' (nom) et 'grade' et une méthode 'toString' personnalisée. Créer un objet 'team' qui contient des membres. Créer une instance de 'team' et y ajouter des membres. Afficher les membres de 'team' en utilisant la fonction 'toString' de 'member'.
